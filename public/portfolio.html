Things I've Built.

★ Featured Project
01 / ML + CV
AI-Based Environmental Audio-to-Visual Accessibility System

An end-to-end environmental sound recognition system that converts audio signals into accessible visual video outputs — designed for accessibility and environmental awareness. Trained on ESC-50 dataset with ~92% cross-validated accuracy.

Python Machine Learning SVM MFCC OpenCV MoviePy Audio Signal Processing
  • MFCC, spectral features, chroma & time-domain feature engineering to transform raw audio into ML-ready representations
  • SVM classifier with RBF kernel trained on ESC-50 dataset — dog, rain, siren, footsteps classification
  • Automated inference pipeline: .wav → classify → generate visual output with labels & confidence scores
  • OpenCV multimedia interface with semantic icons & real-time loudness indicators
  • MoviePy-based audio–video synchronisation for final MP4 output
